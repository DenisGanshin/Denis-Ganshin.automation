# Справочник SVG-иконок сервисов

## Как использовать

### Вариант 1: Simple Icons CDN (не нужно скачивать)
```html
<img src="https://cdn.simpleicons.org/telegram" alt="Telegram" width="24" height="24">
<!-- С кастомным цветом: -->
<img src="https://cdn.simpleicons.org/telegram/000000" alt="Telegram" width="24" height="24">
```

### Вариант 2: Локальные файлы
```html
<img src="img/icons/telegram.svg" alt="Telegram" width="24" height="24">
```

### Примечания
- **icon** = только символ/пиктограмма, компактный, подходит для маленьких размеров
- **logo** = текст + символ, больше подходит для больших блоков
- Mango Office — единственный PNG (SVG с Wikimedia не скачался)
- Для чёрных/белых CDN-иконок: `/telegram/000000` или `/telegram/ffffff`
