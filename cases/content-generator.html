Проблема
  • Проблема чистого листа — сесть писать пост сложнее, чем рассказать мысль голосом
  • Хочешь вести канал регулярно, но вечно не хватает времени — а делегировать не можешь, потому что это же мой личный блог
  • На один пост уходит 30–60 минут, а то и 3 часа — перфекционизм не даёт опубликовать, пока не отточишь каждую формулировку
  • Нужно адаптировать контент под разные площадки — Telegram, Instagram, LinkedIn, Threads
Решение

Telegram-бот, обученный на примерах автора. Надиктовываешь мысль голосовым — получаешь готовый пост в своём стиле.

Генерирует разные текстовые форматы: лонгрид, короткий пост, тред.

Стек технологий

LLM Claude
Голос Web Speech API
Оркестратор n8n
Мессенджер Telegram Bot API
Форматы Посты в Telegram, Threads, Instagram, LinkedIn

Функциональность

Голосовой ввод

  • Транскрибирует голосовые сообщения из Telegram
  • Извлекает ключевую мысль и структуру
  • Работает с русским и английским

Стилизация

  • Подбирает характерные обороты, метафоры, структуру
  • Попадание в авторский стиль

Мультиформат

  • Лонгрид для блога (1000+ слов)
  • Короткий пост для Telegram (200–400 слов)
  • Тред для Threads / X (серия коротких постов)

Редактура

  • Корректирует тон: экспертный, разговорный, провокационный
  • Блок памяти позволяет давать правки и исправлять текст в режиме реального времени

Обучение на примерах

  • Загружаешь 10–20 лучших постов — бот учит твой стиль
  • Анализ: какие паттерны дают больше вовлечённости
