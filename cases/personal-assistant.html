Проблема
  • Постоянное переключение между приложениями — календарь, почта, заметки
  • Задачи забываются, если не записать сразу — а записывать долго
  • Голосовые сообщения удобнее текста, но сервисы не понимают речь
  • Нет единой точки входа — информация разбросана по 5+ приложениям
Решение

Один Telegram-бот, который управляет календарём, почтой, задачами и заметками. Понимает голосовые — расшифровывает и выполняет.

Интегрирован с Google Calendar, Gmail, Perplexity, Bitrix/Notion/Trello. Все команды через один чат — не нужно открывать другие приложения.

Ставится иконкой на рабочий экран — выглядит как ещё одно приложение, но за ним стоит AI.

Функциональность

Календарь и события

  • Добавляет и корректирует мероприятия в Google Calendar
  • Отправляет приглашения участникам через Gmail
  • Расшифровывает голосовые — команда голосом или текстом

Почта из чата

  • Подготавливает и отправляет email, не выходя из Telegram
  • Формулирует текст по контексту разговора
  • Не читает входящие — только исходящие

Задачи и заметки

  • Создаёт задачи в Bitrix, Notion или Trello
  • Записывает заметки в Google Sheets / Airtable — списки покупок, идеи, планы
  • Работает с голосовыми и текстом

Поиск и тексты

  • Ищет информацию в интернете через Perplexity
  • Помогает с текстами — редактура, переформулировка, перевод
  • Личный tutor по изучению языков

Голосовой интерфейс

  • Расшифровывает голосовые сообщения в нормализованный текст
  • Иконка на рабочем экране — работает как приложение
  • Модуль памяти на 20 сообщений для каждого диалога
