Автоматизация холодного аутрича

Каскадная система персонализированных рассылок: Email → LinkedIn → Telegram. Человек подключается только когда получен ответ.

−2 ч рутины/день
×4 охват в день
n8n Airtable Perplexity GPT-4.1-mini Gmail API Unipile
Проблема
  • Каждое сообщение нужно писать отдельно, учитывая контекст компании и человека
  • Нужно помнить, кому писали и когда, чтобы вовремя сделать follow-up
  • При отсутствии ответа на email — вручную переходить в LinkedIn, потом в Telegram
  • При ответе в одном канале — вручную останавливать рассылку в остальных
  • Масштабировать невозможно без потери качества: либо ручная работа, либо шаблонный спам
Решение

AI исследует компанию и человека через Perplexity перед каждым касанием и генерирует уникальное сообщение под контекст: канал, история предыдущих касаний, язык общения.

Каскад автоматически переключает каналы через 3 дня без ответа — Email → LinkedIn → Telegram. Ответ в любом канале мгновенно останавливает всю цепочку.

Встроенные лимиты и случайные задержки между отправками — система ведёт себя как человек, не как бот.

Стек технологий

Оркестрация n8n (self-hosted, Railway)
База контактов Airtable
Исследование компаний Perplexity Sonar API
Генерация текстов GPT-4.1-mini
Email + трекинг Gmail API, pixel tracker
LinkedIn + Telegram Unipile API

Функциональность

Email-рассылка

  • AI-исследование компании через Perplexity — результат кешируется, платим за поиск один раз
  • AI генерирует уникальное письмо под компанию, должность и индустрию
  • 3 автоматических follow-up, если нет ответа — AI знает, какое по счёту касание
  • Трекинг открытий письма в реальном времени через встроенный пиксель

LinkedIn-аутрич

  • Запуск через 3 дня после email или сразу при bounce
  • AI-нота учитывает историю: видит, что письмо открыли, но ответа не было
  • Трекинг ответов через Unipile webhook

Управление каскадом

  • Ежедневная проверка: кому переходить на следующий этап
  • Ответ в любом канале → статус Replied → вся цепочка замораживается
  • Финальный статус: все каналы исчерпаны — система останавливается

Аналитика

  • Дашборды за 7 дней и за всё время в Airtable
  • Ежедневный пересчёт метрик + Telegram-алерт с итогами
  • История всех касаний для каждого лида в поле outreach_log

Мультиаккаунт и мультикампания

  • Таблица Projects — консоль управления кампаниями: цель, язык, лимиты отправок
  • Таблица Senders — справочник аккаунтов-отправителей (email, LinkedIn)
  • Каждый лид привязан к проекту и отправителю
